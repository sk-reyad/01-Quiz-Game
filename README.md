# 🧠 Interactive Tech Quiz Game

A sleek, dynamic, and interactive quiz application built to test your knowledge of Computer Science and Technology. From basic web acronyms to Java programming specifics, this quiz challenges users while providing instant, visual feedback.

## ✨ Features

* **Dynamic Question Loading:** Questions and answers are rendered dynamically using JavaScript, making it incredibly easy to add or modify the question bank.
* **Real-Time Feedback:** Instantly highlights the correct answer in green and incorrect answers in red as soon as the user makes a selection.
* **Animated Progress Tracking:** Features a clean, animated progress bar that updates as you move through the quiz, along with a live question counter.
* **Score & Result Evaluation:** Tracks your score in real-time and provides a custom, dynamic message at the end based on your final percentage (e.g., "Perfect! You're a genius!" vs. "Good effort! Keep learning!").
* **Responsive Design:** Fully optimized with CSS media queries to ensure the game looks great and functions perfectly on both desktop monitors and mobile screens.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure dividing the app into clear Start, Quiz, and Result screens.
* **CSS3:** Custom styling, Flexbox layouts, interactive hover states, visual state indicators (correct/incorrect), and mobile responsiveness.
* **Vanilla JavaScript (ES6+):** Complete game logic including DOM manipulation, event listeners, array iteration, and state management (tracking scores and question indexes).

## 🚀 How to Run Locally

Since this project uses pure Vanilla JavaScript, HTML, and CSS, there is no need for `npm` or complex server setups.
